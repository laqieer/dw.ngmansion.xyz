SkillSystems Custom Build

Build and install SkillSystems from source code.
Use this to edit SkillSystems to add skills, and change other things you would not normally be able to.

This feature is dangerous, so be sure to make a backup before running it.
Also, be sure to test the ROM after running Custom Build.

If something breaks, close FEBuilderGBA without saving. That way you will not put the errors in your project ROM.

What you need to know

You need to understand the syntax of the EA and how to edit it.
You need to understand how to set up SkillSystems.

How to use

Download the SkillSystems Custom Build source code.
https://github.com/FEBuilderGBA/FEBuilderGBA/releases/download/ver_20220712.19/SkillSystem_FE8-Skill20220703.7z

Edit the files related to the changes you want to make. (eg. “Engine Hacks/SkillSystem” for skills, “Engine Hacks/External Hacks” for other hacks, etc.)
Also, be sure to edit “Engine Hacks/Config.event” to change the configurations to your liking.

Skill assignments by default are the same as FEBuilderGBA, so if you want them to be the same, you do not need to change them

Select “MAKE CUSTOM_BUILD.cmd” from “Skill Custom Build” in the advanced menu of FEBuilderGBA, build, and test the ROM to make sure there are no errors. If all is well, save the ROM. If not, close FEBuilderGBA without saving.


If you have read this far and do not understand what was said, we recommend that you do not use Custom Build.
