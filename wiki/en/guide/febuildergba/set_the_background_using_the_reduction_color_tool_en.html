In this section, we will explain how to insert a background using the color reduction tool considering TSA installed in FEBuilderGBA.


What is color reduction considering TSA?

With color reduction considering TSA, you can see that green on green onion has come out properly.
The background of FE can deliver up to 8 palettes * 15 colors, but to do it, TSA has to be considered and it was hard to make it manually.
Therefore, we did not care about TSA, we reduced color to 16 colors and used it.
Therefore, we did not care about TSA, we reduced color to 16 colors and used it.
However, since FEBuilderGBA has a color reduction function that takes TSA into account, it is possible to increase the number of colors more.

Setting method.

StepA1

From the menu “Tools” → “Color Reduction Tool”.

StepA2

The color reduction tool will be displayed.

StepA3

Sets the original file and the save destination of the converted image.

StepA4

Click the start button.
Color reduction processing is started.

StepA5

Let's see the completed image at the edge.
The pallet is allocated properly.

StepA6

Let's just set this image to BG.
Please press “Image setting”.

StepA7

Please select “Background”.

StepA8

Since the background screen appears …

StepA9

Please select the background of the prologue.
With FE8J, ID:0x25.
(Please be aware that ID is different for FE8U.)

StepA10

Please press the “image reading” button and select the reduced color image.

StepA11

The image is loaded.

StepA12

Please start the game.
Two people talking in front of ramen should come out.

Weak point.

Currently, FEBuilderGBA is decreasing color using the K-means method, but this algorithm is vulnerable to gradation.

Pyramid original image.


16 color reduction color.


Reduced color considering TSA with FEBuilderGBA

Unfortunately, the boundary of the gradation did not come out neatly.
In this case, you can either correct it by yourself, based on the image of FEBuilderGBA, or give up with 16 colors subtractive color.
