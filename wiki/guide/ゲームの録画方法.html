guide:ゲームの録画方法

エミュレータでゲームの内容を録画する方法を説明します。
ここでは、TSAビデオのように、キー入力を保存する軽量の方法を紹介します。
ゲームを録画しておくとデバッグやテストに大変役に立ちます。

VBA-Mでの録画方法
MENU → Tool → Record → Start Movie Recording
適当なファイル名をつけて保存
ゲームのプレイがvmvファイルに保存されます。

VBA-Mでの再生方法
MENU → Tool → Play → Start Playing Movie

保存にはセーブデータが含まれるため、簡単に再現できます。
その代り、再生すると既存のセーブデータが上書きされるので注意。

mGBAでの録画法方法
MENU → Audio/Video → Record video
適当なファイル名をつけて保存

mGBAでの再生方法
MENU → File → Load ROM
ここで、mvlファイルを選択します。

または、 mvlをmGBAへドラッグ&ドロップします。


mGBAでの録画法方法2

mGBAでは、起動オプションでファイルを指定できるようだ。
-v, –movie FILE Play back a movie of recorded input
